Hosting Guide

🔒 Where is your data stored?

Currently all data lives in your browser's memory (localStorage). It persists on the same device/browser but is NOT on any server. To share across team members you need a backend.

Recommended Hosting Options

⭐ Option 1 — Netlify (Free, Easiest)

✅ Free Tier 🚀 5 min Deploy 🌍 CDN Included

Drop this HTML file into Netlify Drop at app.netlify.com/drop and your dashboard goes live instantly with a URL you can share.

Data: Add Netlify Blob Storage or connect a Google Sheet via their Functions for persistent data.

⭐ Option 2 — Vercel + Supabase (Best for Teams)

✅ Free Tier 🗄️ PostgreSQL DB 🔑 Auth Built-in

Host this file on Vercel (vercel.com) and use Supabase (supabase.com) as your database. Supabase gives you a free PostgreSQL database with a REST API. Store CSV data in a table, query it with JS.

💡 Supabase free tier: 500MB DB, unlimited API calls — perfect for small teams.

Option 3 — Firebase (Google Cloud)

✅ Free Spark Plan ⚡ Realtime DB

Use Firebase Hosting + Firestore. Upload CSV data to Firestore, and the dashboard fetches it in real time — useful for live sales tracking.

Option 4 — Your Own Server (VPS)

💰 ~$5/mo 🔧 Full Control

Use DigitalOcean, Hetzner, or AWS EC2. Run a Node.js/Python backend, store data in MySQL/PostgreSQL. Upload CSV → parse on server → return JSON to frontend. Most secure for company data.

💡 Quick Recommendation for Your Team

If you just need to share this with a few people right now, host on Netlify Drop and use Google Sheets as a database (free). Upload CSV → paste into Google Sheet → dashboard reads via Sheets API.

For a proper product, go with Vercel + Supabase — both free, both production-grade, with user auth baked in.
